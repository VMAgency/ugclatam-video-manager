# UGC LATAM Video Manager

30-day short-form video content management tool for ugclatam.io.
Built for Chad Smalley, CEO — hosted on GitHub Pages.

## Live App
https://VMAgency.github.io/ugclatam-video-manager

## What's inside
Single file: `index.html` — the complete app.
No dependencies, no build tools, no Node.js required.

## To update
Upload a new version of `index.html` — GitHub Pages deploys in 60 seconds.

## Features
- 30-Day Content Calendar with clickable scripts
- Script editor with inline hook / body / CTA editing
- Status Tracker — Scripted → Filmed → Edited → Posted
- Monthly Program Builder
- Hooks Library & CTA Rotation Strategy
- Export all scripts as .txt filming briefs
